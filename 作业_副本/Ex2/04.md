## (a)题目：年龄段分组统计
```r
ex01BinCounting <- function(numbers, cutoffs, binnames) {
	result <- numeric(length(binnames))   #初始化结果向量
	names(result) <- binnames   #给结果向量命名
	
	for (num in numbers) {
		for (i in 1:length(cutoffs)) {
			if (num < cutoffs[i]) {   #当数字小于该年龄段分割线时
				result[i] <- result[i] + 1   #该年龄段组数量加一
				break   #直到没有属于该年龄段的数字后跳出循环
			}
		}
		if (num >= cutoffs[length(cutoffs)]) {   #对于最后一组年龄段
			result[length(binnames)] <- result[length(binnames)] + 1
		}
	}
	return (result)   #返回结果向量
}
```

## (b)题目：年龄分类标记
```r
ex02Binning <- function(numbers, cutoffs, binnames) {
	cut(numbers, breaks = c(0, cutoffs, Inf), labels = binnames, include.lowest = TRUE, right = FALSE, ordered_result = TRUE)
}
```
* **函数 cut(x, breaks, labels, include.lowest, right, ordered_result):** 
	* **x:** 需要分类的数据
	* **breaks:** 分割线
	* **labels:** 分组标记
	* **include.lowest:** 逻辑值，每组是否要包含左边的临界点
	* **right:** 逻辑值，每组是否要包含右边的临界点（逻辑值应和 include.lowest **相反**）
	* **ordered_result:** 逻辑值，结果是否是有序因子
