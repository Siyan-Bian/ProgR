## 题目："Fizz Buzz"
```r
ex01FizzBuzz <- function(up.to, fizz.number = 3, buzz.number = 5) {
	x <- seq_len(up.to)
	fizzing <- x %% fizz.number == 0
	buzzing <- x %% buzz.number == 0
	
	result <- as.character(x)
	result[fizzing] <- "Fizz"
	result[buzzing] <- "Buzz"
	result[fizzing & buzzing] <- "Fizz Buzz"
	
	return(result)
}

ex01FizzBuzz(10, 3, 5)
ex01FizzBuzz(15, 3, 5)
```
* **`as.character(x)`**: 把数字向量转为字符向量
	* **目的**: 为了后续能够安全地将 "Fizz", "Buzz" 等字符串赋值到向量中
	* **必要性**: R 向量要求元素同类型，字符和数字不能混存，所以统一转为字符型
