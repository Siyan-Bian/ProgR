## (a)题目：向量类型判断
```r
ex01VectorType <- function(x) {
  if (is.logical(x)) {
    print("lg")
  } else if (is.numeric(x)) {
    print("nm")
  } else if (is.character(x)) {
    print("ch") 
  } else {
    print("x")
  }
}

ex01VectorType(2)
ex01VectorType("hello")
ex01VectorType(TRUE)
ex01VectorType(c(1, 2))
```

## (b)返回向量中偶数
```r
ex02Odd <- function(x) {
  x[x %% 2 == 0]
}

ex02Odd(c(1, 2, 3, 4, 5, 6))
```
