## R语言函数运行时间测量函数

##### 函数目标
编写一个函数 `ex01TimeFun`，用于测量另一个函数的运行时间。
##### 输入参数
**1. fn（待测量的函数）**
- 类型：函数
- 特点：无参数函数，调用方式为 `fn()`
**2. impute.inf（错误处理标志）**
- 类型：逻辑值（单个TRUE或FALSE）
- 作用：
    - `TRUE`：当函数出错时，返回 `Inf`
    - `FALSE`：当函数出错时，返回实际运行时间（错误前的时间）
##### 返回值
- 函数的运行时间（秒，四舍五入到整数）
- 如果出错且 `impute.inf = TRUE`，返回 `Inf`

```r
ex01TimeFun <- function(fn, impute.inf) {
  # your code
  assertFunction(fn, nargs = 0)  #检查是否为函数类型
  assertFlag(impute.inf)  #检查参数是否是单一的逻辑值
  #三层嵌套结构
  runtime <- system.time({  #测量表达式执行所需的时间
    return.inf <- tryCatch({
      fn()  #执行函数
      FALSE #执行到这里，返回FALSE
    }, error = function(e) {
      impute.inf  #返回impute.inf的值
    })
  })
  if (return.inf) return(Inf)  #当return.inf = TRUE时，返回Inf
  round(runtime[[3]])  #返回四舍五入的运行时间，提取第三个元素 elapsed（实际经过时间）
}
```
* **assertFunction(X, nargs):**
	* **X:** 检查的对象
	* **nargs:** 参数数量
		* nargs = 0: 验证函数不需要任何参数
		* 示例:
```r
assertFunction(sleep1, nargs = 0)  # ✓ 通过
assertFunction(function(x) x, nargs = 0)  # ✗ 报错，需要1个参数
```
* **assertFlag():** 检查单个逻辑值的函数
	* 等价于`assertLogical(impute.inf, len = 1, any.missing = FALSE)`
	* 允许的值：TRUE; FALSE
	* 不允许的值: 
		* c(TRUE, FALSE) ---长度不为1
		* NA ---缺失值
* **system.time(expr):**
	* expr: 一个R表达式
	* **注意**：如果需要运行的是一个**负责的逻辑**，就必须用`{}`把这些多行代码"打包"成一个单一的表达式
* **trycatch():** 抛出错误函数
	* 返回值是它执行的最后一个表达式的值
		* 正常执行: 返回 FALSE
		* 发生错误: 返回 impute.inf 的值
	* **基本结构:**
```r
tryCatch(
  { 尝试执行的代码 },
  error = function(e) { 错误处理代码 }
)
```
* **提取 elapsed 列**:
	* `runtime[[3]]` 或者 `runtime["elapsed"]`
* **主程序流程:**
	* **情况1: fn()成功执行**
		* 执行fn()
		* 返回FALSE
		* 不执行 impute.inf
	* **情况2: fn()不成功执行，抛出错误**
		* 执行fn()失败，跳转到Error处理器
		* 执行 impute.inf
		* 返回 impute.inf 的值
* **什么时候return.inf = TRUE?**
	* 当 fn() 出错且 inpute.inf = TRUE 时
