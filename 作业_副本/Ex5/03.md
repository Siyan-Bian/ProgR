## 题目: R语言安全版 lapply 函数

##### 函数目标
编写一个类似 `lapply()` 的函数，但具有错误处理能力：当应用的函数出错时，不中断执行，而是用指定的值替代。
##### 输入参数
**1. X（待处理的对象）**
- 类型：任何 `lapply()` 可以迭代的对象
    - 向量：`c(1, 2, 3)`
    - 列表：`list(1, "a", TRUE)`
    - 等等
- 无需验证
**2. FUN（要应用的函数）**
- 类型：函数
- 特点：接受一个参数
- 需要验证：`assertFunction(FUN, nargs = 1)`
**3. impute（替代值）**
- 类型：任意类型
    - 可以是数字：`-1`
    - 可以是 `NULL`
    - 可以是列表、字符串等任何值
- 无需验证

```r
ex01SafeLapply <- function(X, FUN, impute) {
  # your code
  assertFunction(FUN, nargs = 1)
  lapply(X, function(x) {
    tryCatch(FUN(x), error = function(e) impute)
  })
}
```
