## (a)字符串拼接
* Examples:
	* parent = "Herbert", male = TRUE, child = "Klaus Dieter" --> `Herbert is the father of "Klaus Dieter".`
	* parent = "Hildegard", male = FALSE, child = "Adelheid" --> `Hildegard is the mother of "Adelheid".`
	* parent = "Y", male = FALSE, child  = "A"     --> `Y is the mother of "A".`
```r
ex01Children <- function(parent, male, child) {
  # your code
  #参数验证
  assertString(parent)  #确保是单个字符串
  assertFlag(male)  #确保是单个逻辑值
  assertString(child)  #确保是单个字符串
  
  #字符串格式化
  sprintf("%s is the %s of \"%s\".", parent, if (male) "father" else "mother", child)
}
```
* " \ " 作用: 转义符号，为了显示出 child 周围的 ""


## (b)拆解句子为列表
* Examples:
	* 'Eric is the father of "Bob".' --> list(parent = "Eric", male = TRUE, child = "Bob")
	* 'Herbert is the father of "Klaus Dieter".' --> list(parent = "Herbert", male = TRUE, child = "Klaus Dieter")
	* 'Gudrun is the mother of "Rosamunde ("Rosi")".' --> list(parent = "Gudrun", male = FALSE, child = 'Rosamunde ("Rosi")')
```r
ex02ChildrenInverse <- function(sentence) {
  # your code
  assertString(sentence)
  #regexec 返回匹配的位置和长度
  m <- regexec("^(.*) is the (mother|father) of \"(.*)\"\\.$", sentence)
  #regmatches 提取实际匹配的文本
  found <- regmatches(sentence, m)[[1]][-1]
  #构建返回列表
  list(parent = found[[1]], male = found[[2]] == "father", child = found[[3]])
}
```
* **正则表达式拆分详解**

| 部分               | 含义                       | 捕获内容   |
| ---------------- | ------------------------ | ------ |
| ^                | 字符串开始                    | -      |
| (.*)             | **捕获组1**：任意字符（父母名字）      | parent |
| is the           | 字面文本                     | -      |
| (mother\|father) | **捕获组2**：mother 或 father | 判断性别   |
| of               | 字面文本                     | -      |
| \ "              | 转义的双引号（字面字符 `"`）         | -      |
| (.*)             | **捕获组3**：任意字符（孩子名字）      | child  |
| \ "              | 转义的双引号（字面字符 `"`）         | -      |
| \ \ .            | 转义的句号（字面字符 `.`）          | -      |
| $                | 字符串结束                    | -      |
* **`regexec` 和 `regmatches` 的工作原理**
```r
sentence <- 'Herbert is the father of "Klaus Dieter".'

# regexec 返回匹配位置和长度
m <- regexec("^(.*) is the (mother|father) of \"(.*)\"\\.$", sentence)
# m 是一个列表，包含匹配的起始位置和长度

# regmatches 提取实际匹配的文本
regmatches(sentence, m)
# [[1]]
# [1] "Herbert is the father of \"Klaus Dieter\"."  # 完整匹配
# [2] "Herbert"                                      # 捕获组1
# [3] "father"                                       # 捕获组2
# [4] "Klaus Dieter"                                 # 捕获组3

# [-1] 移除第一个元素（完整匹配）
found <- regmatches(sentence, m)[[1]][-1]
# [1] "Herbert"      "father"       "Klaus Dieter"
```
