## (a)题目：返回矩阵中每行第一个非零数

```r
ex01RowFirsts <- function(mat) {
  # your code
  assertMatrix(mat, mode = "numeric", any.missing = FALSE)   #检查输入是否为矩阵
  result <- apply(mat, 1,function(row) {   #构建函数代入参数
    which(row != 0)[1]   #返回第一个TRUE的索引
  })
  return(result)
}
```
* **apply(X, MARGIN, FUN)):**
	* 参数：
		* **X:** 矩阵
		* **MARGIN:** 1 表示按行操作，2 表示按列操作
		* **FUN:** 对每一行/列应用的函数
	* 工作原理：
		* apply会遍历矩阵的每一行/列
		* 将每一行/列作为向量传递给后面的函数
		* 收集所有行/列的结果并返回一个向量
* **核心逻辑：** which(row != 0)[1]
	* **步骤一：** row != 0
		* 示例：row = c(0, 1, 0) --> 输出：c(FALSE, TRUE, FALSE)
	* **步骤二：** which()
		* 示例：which(c(FALSE, TRUE, FALSE, TRUE)) --> 输出：c(2, 4)
		* which()返回逻辑向量中 TRUE 的索引位置 
		* 如果有多个 TRUE，返回所有位置
	* **步骤三：** [1]
		* 示例：which(c(FALSE, TRUE, FALSE, TRUE))[1] --> 输出：2

## 提取数据框对角线元素

```r
ex02TableDiag <- function(df) {
  # your code
  #检查输入是否为data.frame
  assertDataFrame(df)
  
  #确定对角线长度
  diag_length <- min(nrow(df), ncol(df))
  
  #返回一个列表
  lapply(seq_len(diag_length), function(i) df[i,i])
}
```
* **步骤一：** 检查输入类型
	* **assertDataFrame(df):** 检查df是否为数据框
* **步骤二：** 确定对角线长度
	* nrow(): 数据框的行数
	* ncol(): 数据框的列
	* min(): 取两者中的最小值
	* **为什么取最小值？**
		* 如果是 3×5 的数据框，只有前3个对角线元素（(1,1), (2,2), (3,3)）
		- 如果是 5×3 的数据框，也只有前3个对角线元素
- **步骤三：** 生成从1到对角线长度的序列
	- **seq_len(n):** 生成从1到n的整数序列 
		- 例如: seq_len(5) ---> c(1, 2, 3, 4, 5)#
		- **和 1:n 的区别:**
			- n <- 0: 
				- 1:0 ---> c(1, 0)
				- seq_len(0) ---> integer(0)
			- n <- 3:
				- 1:3 ---> c(1, 2, 3)
				- seq_len(3) ---> c(1, 2, 3)
			- n <- -2:
				- 1:-2 ---> c(1, 0, -1, -2)
				- seq_len(-2) ---> Error
		- **和seq_along()的区别:**
			- seq_along(x)作用: 生成沿着对象x长度的索引序列
				- x: 任意向量或列表
				- 返回 1: length(x) 的整数列表
				- 例如: 
					- x <- c("a", "b", "c") --- seq_along(x) ---> c(1, 2, 3)
					- y <- list(10, 20, 30) --- seq_along(y) ---> c(1, 2, 3)
				- 和 1:length(x) 的区别:
					- x <- character(0) :
						- seq_along(x) ---> integer(0)
						- 1:length(x) ---> c(1, 0)
		- **优势**
			- 安全处理 n=0
			- 负数会报错而并非产生奇怪结果
- **步骤四:** 匿名函数(没有名字的函数，临时创建，通常用于lapply，sapply等高级函数)
	- **此处作用:** data.frame索引
- **步骤五:** lapply() 对列表每个元素应用函数
	- 目的: 题目要求返回**list**，lapply()天然返回**list**
