## (a)题目: 将多个字符串收集并合并成一个字符串
```r
ex01Collate <- function(...) {   #接受多个参数，使用...
  # your code
  #收集所有参数
  args <- list(...)
  #验证每个参数都是标量字符串
  lapply(args, assertString)
  #为每个字符串添加单引号
  quoted <- sprintf('"%s"', unlist(args))
  #用逗号拼接
  paste(quoted, collapse = ", ")
}
```
* **function(...):** 
	* 接受任意数量的参数
	* 参数可以是任意类型
	* 常用于需要灵活参数的函数
* **list(...):** 
	* 将 ... 中所有参数收集到一个列表中
* **sprintf(fmt, ...):**
	* **fmt:** 格式化字符串(包含占位符)
		* 常用占位符: 
			* %s: 字符串
			* %d: 整数
			* %f: 浮点数
			* %.2f: 保留2位小数的浮点数
	* **... :** 要插入的值
	* **返回:** 格式化后的字符串
* **unlist():** 将列表转换为向量
	* 使用原因: sprintf() 期望向量，不是列表
* **paste():**
	* 模式1: 折叠向量为单个字符串(使用collapse)
		* 例如: paste(c("a", "b", "c"), collapse = ",") ---> "a, b, c"
	* 模式2: 连接多个参数(使用sep)
		* 例如: paste("Hello", "World", sep = " ") ---> "Hello World"
	* `sep` 和 `collapse` 同时使用
		* `paste(c("A", "B"), c("1", "2"), sep = "-", collapse = " & ")`
		* 返回: `"A-1 & B-2"`


## (b)题目: 多次调用 ex01Collate 函数
```r
ex02CollateLists <- function(inputs) {
  # your code
  #验证输入是字符向量的列表
  assertList(inputs, types = "character", any.missing = FALSE)
  #对每个字符向量调用 ex01Collate
  vapply(inputs, function(inp) {
    do.call(ex01Collate, as.list(inp))
  }, "")
}
```
* **vapply(X, FUN, FUN.VALUE):** 对列表每个元素应用函数
	* **X:** 要处理的向量或列表
	* **FUN:** 应用到每个元素的函数
	* **FUN.VALUE:** 函数返回值的模版 (这里 FUN.VALUE = "" 表示返回单个字符串， 等价于character(1))
* **as.list():** 将向量转换为列表
	* 为什么使用该函数? ---向量不能直接用于 do.call()，会报错
* **do.call(what, args):** 将列表元素作为参数调用函数
	* **what:** 要调用的函数
	* **args:** 参数列表(必须是列表)
