## (a)DataTable 去重
* **输入:** `table`：一个包含列 `year`、`month`、`day` 的 `data.table`
* **输出**：输入数据表中根据`year`、`month`、`day`列移除重复项后的结果
```r
ex01DedupTable <- function(table) {
  # your code
  assertDataTable(data)
  assertSubset(c("year", "month", "day"), names(table))
  unique(table, by = c("year", "month", "day"), fromLast = TRUE)
}
```



## (b)分组聚合
```r
ex02LastResponse <- function(response) {
  # your code
  assertDataTable(response) 
  response[order(submission.date),  # 第1步：排序
           lapply(.SD, function(x) last(na.omit(x))),  # 第2步：取最后非NA值
           by = name][  # 第3步：按姓名分组
             submission.date := NULL   # 第4步：删除列
           ]
}
```
