## 题目: 提取文本中所有大写开头但不在句首的单词

```r
ex01ProperNoun <- function(text) {
  # your code
  #参数验证
  assertString(text)
  
  #定义正则表达式
  pattern <- "(?<!\\.) [A-Z][[:alnum:]]*"

  #regmatches返回列表, [[1]]提取实际的字符向量
  matches <- regmatches(text, gregexpr(pattern, text, perl = TRUE))[[1]]

  #去除空格并去重
  unique(trimws(matches))
}
```
* **assertString()**: 检查“一个”字符串
	* 和 **assertCharacter()** 的区别:
		* assertString() 检查对象必须是**长度为 1** 的字符型向量: 仅 `"A"`
		* assertCharacter() 检查对象必须是字符型**向量**: `c("A", "B", "C")` 或 `"A"`
* **正则表达式拆解分析:**
	* `(?<!\\.)`: 匹配位置的**前一个字符不能是点**
		* `?<!`: 负向后顾断言，检查匹配位置**之前**的内容，但**不包含**在匹配结果中
		* `\\.`：匹配**字面意义的点**
			* `.` 是**特殊字符**，匹配任意字符
			* `\` 本身需要转义为 `\\`
	* ` [A-Z]`: 
		* 空格 ` `: 确保单词前面有空格（即不在句首）
		* `[A-Z]`: 匹配**一个**大写字母（A到Z）
	* `[[:alnum:]]*`: 匹配**零个或多个**字母数字字符
		* `[[:alnum:]]`: **POSIX字符类：** 匹配**字母或数字**
			* **等价于：** `[A-Za-z0-9]`
			* **匹配:** 字母: A-Z, a-z, 数字: 0-9
		* `*` 量词: 零个或多个
* **gregexpr(pattern, text):** 提取**所有**符合条件的单词，而不只是第一个
	* 为什么需要 `perl = TRUE`？
		* 后顾断言 `(?<!...)` 是**Perl扩展语法**，不加会报Error
	* 和 `regexpr()` 的区别:
		* regexpr: 每个元素找第一个匹配
			* regexpr(pattern, text) ---> 返回: 整数向量（位置）
		* gregexpr: 每个元素找所有匹配
			* gregexpr(pattern, text) ---> 返回: 列表，每个元素是一个整数向量
		* **示例：**
```r
text <- "Hello World is Great"

# regexpr（只找第一个）
m1 <- regexpr(" [A-Z]", text)
m1
# [1] 6
# attr(,"match.length") [1] 2
# 只找到 " W"

# gregexpr（找所有）
m2 <- gregexpr(" [A-Z]", text)
m2
# [[1]]
# [1] 6 15
# attr(,"match.length") [1] 2 2
# 找到 " W" 和 " G"
```
* **trimws():** 去除前后空格
* **unique():** 去重
