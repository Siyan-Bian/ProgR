## (a)计算矩阵中所有"黑色方格"位置的元素之和（类似国际象棋棋盘）

**棋盘规则**：
- 左上角 (1,1) 是白色
- 相邻方格颜色相反
- **黑色方格**：行号 + 列号 = 奇数
```r
ex01ChessboardSum <- function(mat) {
  # your code
  sum(mat[(row(mat) + col(mat)) %% 2 == 1])
}
```
* **含义：**
	* `row(mat)` 和 `col(mat)` - 获取行列索引矩阵
	* `row(mat) + col(mat)` - 计算行列和
	* `%% 2 == 1` - 判断奇偶性
	* `mat[...]` - 提取黑色方格元素
	* `sum()` - 求和


## (b)找到矩阵中最小元素的坐标 `c(行, 列)`
```r
ex02MatrixWhichMin <- function(mat) {
  # your code
  #方法一
  wm <- which.min(mat)   #找到最小值的位置索引
  as.numeric(c(row(mat)[wm], col(mat)[wm]))
  
  #方法二
  wm <- which.min(mat)
  as.numeric(arrayInd(wm, dim(mat)))
}
```
##### 方法一
* **which.min():** 返回矩阵**展平为向量**后的索引，不是坐标
* `row(mat)[wm]`, `col(mat)[wm]`: 从行列索引矩阵中提取对应位置
* **as.numeric(c()):** 合并成向量并转换为数值

##### 方法二
* **arrayInd():** 将线性索引转换为多维数组的坐标
