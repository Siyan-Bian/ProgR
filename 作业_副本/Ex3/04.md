## 题目：每一步中，如果当前位置或其右邻位置值为 1，则

```r
x01CellularAutomaton <- function(initial.state) {
  # your code here
  results <- list(initial.state)   #创建列表存储每一代状态，第一个是输入的初始状态
  state <- initial.state   #设置一个工作变量，存储当前迭代的状态
  
  repeat {   #创建无限循环
	  state.shifted <- c(0, head(state, -1) #去掉最后一个元素，并在前面添加0
	  state.next <- ifelse(state == 1 | state.shifted == 1, 1, 0)
	  if (all(state.next == state)) break #检查每个位置是否相同，如果为真则跳出
    results[[length(results) + 1]] <- state <- state.next
  }
  results
}
```
* `length(results) + 1` 计算新的索引位置
- `state <- state.next` 更新当前状态
- `results[[...]] <- ...` 将新状态添加到列表
