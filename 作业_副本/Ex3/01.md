## (a)题目：判断一个字符串是否为回文（palindrome）

```r
ex01Palindrome <- function(input) {
  # your code
  str <- strsplit(toupper(input), "")[[1]] #strsplit返回列表，[[1]]返回字符串
  #class(str) --> "character"
  str <- str[str %in% LETTERS]
  all(str == rev(str))   #检查全部相等为回文
}
```
* **toupper(x)**: 字符串全部转换为大写
* **strsplit(x, split,...)**: 按 split 切分字符串(""空字符串 = 每个字符分割)
* **为什么加 `[[1]]`?** 
	* strsplit() 总是返回一个列表，即使只有一个字符串输入
	* `[[1]]`提取向量
	* 为什么不使用  `[1]`? :  `[1]`仍然返回一个列表


## (b)题目：找到大于输入数字 `n` 的最小回文数
```r
ex02NextPalindrome <- function(n) {
  # your code
  repeat {
    n <- n + 1   #从 n+1 开始检查，确保结果 > 输入
    n.split <- strsplit(as.character(n), "")[[1]]   #转换为字符串并分割
    if (all(n.split == rev(n.split))){    #检查全部相等
	    return(n)
    }
  }
}

#> ex02NextPalindrome(100) --> 101
#> ex02NextPalindrome(4243) --> 4334
#> ex02NextPalindrome(1) --> 2
#> ex02NextPalindrome(0) --> 1
#> ex02NextPalindrome(11) --> 22
```
* **repeat()**: **无限循环**结构，必须通过`break` 或 `return` 来退出
	* 使用repeat()的原因：不知道**迭代次数**(否则**for**循环)，没有**条件**(否则**while**循环)
