## 题目: 从数据框(data)中选出指定数据类型(type)的所有列

**方法一**
```r
ex01SelectDF <- function(data, type) {
  # your code
  include <- numeric(0)   #初始化数值向量
  for(i in 1:length(data)) {   #循环遍历
    col <- data[[i]]   #设置存放列的变量；使用[[]]可以直接提取列元素，没有列名
    if (type %in% class(col)) {   #检查左侧值是否在右侧向量中
      include[[length(include) + 1]] <- i   #向量最新元素为判断完的列的索引值，也就是第几列
    }
  }
  return(data[include])
}
```

**方法二**
```r
ex01SelectDF <- function(data, type) {
  sel.col <- vapply(data, function(col) {
    type %in% class(col)
  }, TRUE)
  return(data[sel.col])   #iris[c(TRUE, TRUE, TRUE, TRUE, FALSE)]
}
```
* **vapply(X, FUN, FUN.VALUE...) 用法：**
	* **X:** 要遍历的对象
	* **FUN:** 应用到每个元素的函数
	* **FUN.VALUE:** 返回值的类型和长度模版
		* 这里是 TRUE，说明要求每次返回一个逻辑值
		* 其他常见用法：
			* 返回数值向量的第一个元素：vapply(list(1:3, 4:6), mean, 0)
			* 返回字符向量的第一个元素：vapply(list("a", "bb", "ccc"), nchar, 0)
* **数据框子集提取：**
	* 元素对应的位置为 `TRUE` 时，保留该列
	* 元素对应的位置为 `FALSE` 时，删除该列
