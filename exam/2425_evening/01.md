生成等比数列公式：第 k 项 =$ratio^{k−1}$

```r
geometricSequence <- function(ratio, n) {
  # your code
  checkmate::assert_numeric(ratio)
  checkmate::assert_number(n, lower = 1)
  
  if (n == 0) {
    return(numeric(0))
  }
  
  ratio ^ (0: (n-1))
}
```
* 第1项: 指数0 => ratio⁰ = 1
* 第2项: 指数1 => ratio¹ = ratio
* 第3项: 指数2 => ratio²
* 第n项: 指数n-1 => ratio^(n-1)
