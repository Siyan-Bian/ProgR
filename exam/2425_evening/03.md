编写一个函数 `functionRetry`，该函数尝试将给定函数 `f`
依次应用于向量 `x` 的元素（从 `x[1]` 开始），在首次成功尝试时停止并返回其结果。若 `x` 中所有元素均失败，函数应返回 `NA`。

```r
functionRetry <- function(f, x) {
  # your code
  checkmate::assert_function(f, nargs = 1)
  checkmate::assert_numeric(x, min.len = 1)
  
  for (i in seq_along(x)) {
    result <- tryCatch({
      f(x[i])
    }, error = function(e) {
      NULL  # 用 NULL 区分"函数成功"和"函数失败"
    })
  }
  
  
  if (!is.null(result)) {
    result
  }
  
  NA
}
```
