返回一个矩阵，保留**不是全 NA** 的行

```r
removeMissingRows <- function(mat) {
  # your code
  checkmate::assert_matrix(mat)
  
  if (ncol(mat) == 0) {
    return(matrix(numeric(0), nrow = 0, ncol =0))
  }
  
  keep <- apply(mat, 1, function(row) {
    !all(is.na(row))
  })
  
  mat[keep, , drop = FALSE]
}
```
* `apply(mat, 1, ...)`: 对每一行执行函数
* `is.na(row)`: 哪些元素是 NA
* `all(is.na(row))`: 这一行全是 NA 吗？
* `!all(...)`: 取反: 不是全 NA → TRUE（保留）
