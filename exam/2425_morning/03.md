执行函数 f ，执行成功则返回最大能使 f 执行的 limit 中的值 ，否则抛出错误

```r
getMaxFeasibleInput <- function(f, limit) {
  # your code
  checkmate::assert_function(f)
  checkmate::assert_int(limit, lower = 1)
  
  result <- NA
  
  for(val in seq_len(limit)) {
    tryCatch({
      f(val)
      result <- val  # 没报错，更新结果
    }, error = function(e) {
      # 报错了，什么都不做
    })
  }
  result  # 最后返回记录的最大值（因为是顺序遍历，后面的会覆盖前面的）
}
```
