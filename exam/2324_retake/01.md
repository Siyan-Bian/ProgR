**判断闰年的函数**
```r
getLeapYears <- function(from, to) {
  checkmate::assert_number(from, lower = 1582)
  checkmate::assert_number(to, lower = from)
  
  result <- numeric(0)
  
  for (year in seq(from = from, to =to)) {
    if (year %% 100 == 0 & year %% 400 != 0) {
      next
    }
    
    if (year %% 4 == 0) {
      result <- c(result, year)
    }
  }
  
  result
}
```

**更优方案**
```r
getLeapYears <- function(from, to) {
  checkmate::assert_number(from, lower = 1582)
  checkmate::assert_number(to, lower = from)
  
  years <- seq(from = from, to = to)
  
  # 向量化判断，一行搞定
  years[(years %% 4 == 0 & years %% 100 != 0) | (years %% 400 == 0)]
}
```
