使用自定义比较函数`comparator(x, y)`从给定的字符向量`xs`中识别出“最大”元素
comparator(x, y) 是一个比较大小的函数
* x > y: 返回1
* x < y: 返回-1 
* x == y: 返回0

```r
findMaxFromComparator <- function(xs, comparator) {
  # your code
  assertCharacter(xs, any.missing = FALSE, min.len = 1)
  assertFunction(comparator, nargs = 2)
  
  # 初始化：假设第一个元素是最大值
  x.max <- xs[[1]]
  
  # 遍历剩余元素
  for (i in seq_along(xs)[-1]) {
	# 比较当前元素与当前最大值
    result <- comparator(xs[i], x.max)
    
    # 如果当前元素更大，更新最大值
    if (result == 1) {
      x.max <- xs[i]
    }
  }
  x.max
}
```
