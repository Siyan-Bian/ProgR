编写一个名为 filterRng 的函数，该函数接受一个函数列表作为输入。给定列表中的每个函数均可不带参数调用，且可能使用 R 的随机数生成器（RNG），例如通过调用 runif()、sample()、rnorm() 或类似函数实现。

filterRng应返回仅包含在执行过程中不使用RNG的函数的列表。该函数可对fs中的函数进行任意次数调用，且可假设除可能使用RNG外不存在其他副作用。特别地，这些函数不调用set.seed()。

结果列表中的函数应按其在输入中的出现顺序排列。

### 核心思路

R 的随机数生成器会改变全局变量 `.Random.seed`。如果调用函数前后 `.Random.seed` 没变，说明函数**没有使用 RNG**

```r
filterRng <- function(fs) {
  # your code
  assertList(fs, types = "function")
  runif(1)
  Filter(function(f) {
    old_seed <- .Random.seed
    f()
    !identical(.Random.seed, old_seed)
  }, fs)
}
```
* **理解 `.Random.seed`**
```r
# 初始化 RNG
runif(1)

# 查看当前种子
.Random.seed[1:5]
#> [1]  403  624  ...

# 再次使用 RNG
runif(1)

# 种子改变了
.Random.seed[1:5]
#> [1]  403  624  ...  # 数值不同了
```
