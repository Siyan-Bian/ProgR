## Ex4：字符串拼接
**Return value:**
The return value should be `"Hello, <name>"`
```r
greet <- function(name) {
  # your code
  return (paste0("Hello, ", name))
}
```
* **paste0()**: 没有分隔符
	* greet("Helmut") ----- "Hello, Helmut"
* **paste()**: 默认用空格分隔
	* greet("Helmut") ----- "Hello,  Helmut"

## Ex5: 循环打印
```r
multiGreet <- function(n) {
  # your code
  if (n > 0) {
    for (i in 1:n) {
    cat("Hello\n")  # \n 是换行符
    }
  }
  return (n)
}
```
* `\n`：换行符，每次循环打印一行 "Hello"
* 为什么用 `cat()` 而不是 `print()`？
```r
# cat() 的输出（干净）
cat("Hello\n")
cat("Hello\n")
# Hello
# Hello

# print() 的输出（带引号）
print("Hello")
print("Hello")
# [1] "Hello"
# [1] "Hello"
```

## Ex6：日期天数转换
- **输入**: 月份名称 + 日期
- **输出**: 这一天是一年中的第几天
- **限制**: 不能使用日期相关函数，假设非闰年（2月28天）
```r
dayNumber <- function(month, day) {
  # your code
  # 储存每月天数（命名向量）
  days_in_month <- c(
    January = 31,
    February = 28,
    March = 31,
    April = 30,
    May = 31,
    June = 30,
    July = 31,
    August = 31,
    September = 30,
    October = 31,
    November = 30,
    December = 31
  )
  
  month_index <- match(month, names(days_in_month))  #找到月份位置(match())
  
  # 计算前面月份的总天数（sum()）
  if (month_index == 1) {
    return(day)
  } else {
    days_before <- sum(days_in_month[1:(month_index - 1)])
    return(days_before + day) 
  }
}
```
* **match():**
	* 语法：match(x, table)
	* 返回：x 在 table 中第一次出现的位置
* 为什么不用 `which()`？ -----which() 需要逻辑条件
	* z.B. `which(names(days_in_month) == "March")`
