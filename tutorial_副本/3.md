## Ex2
##### (b)对于每个"words"中的单词，在"text"中出现了几次
* 两个参数 "words", "text" 都是字符串向量
* 结果返回一个数值向量
* 使用 **unname()** 函数
```r
countWordOccurrences <- function(words, text) {
  # your code
  # 验证参数
  checkmate::assert_character(words)
  checkmate::assert_character(text)
  
  # 确定结果向量类型
  result <- numeric(length(words))
  
  # 循环遍历
  for (i in seq_along(words)) {
    count <- 0
    
    for (j in seq_along(text)) {
      if (words[i] == text[j]) {
        count <- count + 1
      }
    }
    result[i] <- count
  }
  return(unname(result))
}
```
* **seq(from, to, by, length.out):** 生成自定义序列
	* from: 起点
	* to: 终点
	* by: 步长
	* length.out: 指定长度
	* **注意:** `seq(n)` 在 `n` 可能为 0 时会出问题 ----- seq(0) 返回 1 0 (使用seq_len(0))
* **seq_along(x):** 索引序列生成器
	* `seq_along(x)` 生成 `1:length(x)`
```r
x <- c("apple", "banana", "cherry")
seq_along(x) 
# [1] 1 2 3
# 等价于（但更安全） 
1:length(x) 
# [1] 1 2 3
```
* **seq_len(n):** 生成 `1:n`，但在 `n = 0` 时更安全
	* seq:len(5) ----- 1 2 3 4 5
	* 比 `1:n` 更安全: 
		* n <- 0
		* 1: n ----- 1 0 => 错误！
		* seq_len(n) ----- integer(0) => 正确！空向量

##### (c)矩阵中每行含有多少个 NA
* 输入参数为 matrix
* 返回结果为数值向量
```r
countNAsPerRow <- function(mat) {
  # your code
  checkmate::assert_matrix(mat)
  
  result <- apply(mat, 1, FUN = function(row) {
    sum(is.na(row))
  })
  return (result)
}
```
