## Ex1: 匹配一整个单词
* 使用 `[[:alpha:]]` 和单词边界 `"\\b"`
```r
pattern.word <- "\\b[[:alpha:]]+\\b"
```
* 单词边界 `"\\b"`, 它确保匹配的内容是一个独立的“词”
* 不用 `"\\b"` 的正则 hi: 他会匹配"hi", "hi"story,t"hi"s, s"hi"p, ...
* 用 `"\\b"` 的正则 hi: 他会匹配"hi there", "say hi!"

## Ex2: 匹配偶数长度的单词
```r
pattern.word.even <- "\\b([[:alpha:]]{2})+\\b"
```

## Ex3: 匹配首尾相同字母的单词
```r
pattern.word.surrounded <- "\\b([[:alpha:]])[[:alpha:]]*\\1\\b"
```
* `\\1` 是一个反向引用，强制要求此处匹配的字母必须与第一部分完全相同，**首尾都已经被定义**，所以中间可以是0次或多次

## Ex4: 匹配逻辑值
* 使用锚点 `^` 和 `$`
```r
pattern.logical <- "^(TRUE|FALSE)$"
```
* 如果你想检查一个字符串是否完全符合某种格式（比如手机号、邮编、逻辑值），必须使用 `^` 和 `$`
* 如果你想在长文本里搜寻某些片段，通常不用 `^` 和 `$`，而是 `\\b`

## Ex5: 匹配向量
* 匹配 `"c(TRUE,FALSE,FALSE,TRUE,...)"`
```r
pattern.logical.vect <- "^c\\((TRUE|FALSE)(,(TRUE|FALSE))*\\)$"
```

## Ex7: 作者名字格式转换
```r
author.pattern <- "^([[:alpha:]])[[:alpha:]]*\\s([[:alpha:]\\s]+)$"  # change this?
author.replacement <- "\\2, \\1."
```
* **捕获组1**: `^([[:alpha:]])` 抓取 First Name 的**第一个字母**
* `[[:alpha:]]*`: 匹配 First Name 剩下的字母（这些字母不需要保留，所以没加括号）
* `\\s`: 匹配名字和姓氏之间的空格
* **捕获组2**: `([[:alpha:]\\s]+)$` 取 Last Name 的**所有内容**
* **替换模式：** `"\\2, \\1."`
	* **`\\2`**：把刚才抓到的“捕获组 2”（整个 Last Name）放到最前面
	- **`,`** ：手动插入逗号和空格
	- **`\\1`**：把刚才抓到的“捕获组 1”（First Name 的首字母）放在后面
	- **`.`**：手动补上句号
- **结果：** `"Albert Einstein"` → `"Einstein, A."`
